## 使用方式
### 启动

#### 步骤 1：启动 embedding（Metal GPU）

```shell
 ./scripts/start-embedding-mac. sh --daemon
```

脚本会自动：创建 venv → 安装 PyTorch MPS + infinity-emb → 检测 Metal 可用性 →
 启动服务 → 健康检查

#### 步骤 2：启动其余服务

 docker compose -f docker-compose. yml -f docker-compose. mac. yml up

### 停止
```shell
docker compose -f docker-compose. yml -f docker-compose. mac. yml down
 ./scripts/stop-embedding-mac. sh
```
